## 🚀 Project Vision

COVIDApp was developed to help users quickly access reliable and up-to-date COVID-19 statistics from around the world. The app provides a clean and simple interface where users can monitor global cases, search country-specific reports, and visualize trends through charts.

This project reflects practical experience in Flutter mobile app development and external API integration.

---

## 🎯 Key Features

### 🌍 Global Statistics Dashboard
- View worldwide COVID-19 statistics instantly.
- Displays:
    - Total Cases
    - Deaths
    - Recovered
    - Active Cases
    - Critical Cases

### 🔍 Country Search Statistics
- Search any country by name.
- View individual country COVID reports in real-time.

### 📊 Charts & Visual Analytics
- Pie charts / graphical representation of data.
- Easy-to-understand data visualization.

### 🎨 Modern User Experience
- Clean UI design
- Smooth navigation
- Responsive layouts for multiple screen sizes

### ⚡ Fast API Integration
- Real-time data fetched using REST API.
- Efficient state updates and smooth performance.
